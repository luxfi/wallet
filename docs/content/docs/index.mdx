# Lux Wallet

An open source multi-chain crypto wallet supporting Web, Desktop, iOS, Android, and Browser Extensions.

## Features

- **Multi-Platform**: Web, Desktop (macOS, Windows, Linux), Mobile (iOS, Android), Browser Extension
- **Multi-Chain**: Support for multiple blockchain networks
- **Open Source**: Fully open source and auditable
- **Hardware Wallet Support**: Integration with hardware wallets
- **DeFi Ready**: Built-in DeFi integrations

## Quick Start

### Prerequisites

1. Install [Node.js LTS](https://nodejs.org/)
2. Install [Yarn](https://yarnpkg.com/)
3. Install [Git LFS](https://git-lfs.github.com/)
4. For iOS: XCode >= 13.3
5. For Android: JDK >= 11

### Installation

```bash
# Clone the repository
git clone https://github.com/luxfi/wallet.git
cd wallet

# Install dependencies
yarn
```

### Development

```bash
# Web development (port 3000)
yarn app:web

# iOS development
yarn app:ios

# Android development
yarn app:android

# Desktop development
yarn app:desktop
```

## Architecture

```
wallet/
  apps/           # Application targets
    web/          # Web application
    desktop/      # Desktop application (Electron)
    mobile/       # iOS and Android (React Native)
    ext/          # Browser extension
  packages/       # Shared packages
    core/         # Core wallet logic
    ui/           # UI components
    sdk/          # SDK utilities
  development/    # Development utilities
```

## Security

- Never share your seed phrase or private keys
- Always verify transaction details before signing
- Report security vulnerabilities to security@lux.network
